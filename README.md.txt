# E-commerce Customer Data Pipeline

This project builds a simple data engineering pipeline using Python, pandas, and SQLite.

## Pipeline Overview

- **Extract**: Load raw customer data from CSV.
- **Transform**: Clean data, add region mapping.
- **Load**: Save cleaned data to SQLite database.
- **Query**: Run SQL queries for business insights.

## Technologies Used

- Python (pandas, sqlite3)
- Jupyter Notebook
- SQLite
- Git + GitHub

## How to Run

1. Open `E commerce.ipynb`.
2. Follow the notebook steps to load, clean, and transform the data.
3. Check the generated SQLite database `ecommerce.db` (if included).
4. Explore queries and visualizations.
